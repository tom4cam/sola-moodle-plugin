# Credits

The `local_ai_course_assistant` Moodle plugin (internally branded "SOLA" — Saylor Online Learning Assistant) was originally built by **Tom Caswell** and **David Ta** at **Saylor University** and open-sourced under GPLv3 for any institution to deploy and brand.

## Core authors
- Tom Caswell (tom.caswell@saylor.org) — original design, features, product direction
- David Ta (david.ta@saylor.org) — infrastructure, deployment, operations

## Origin
Developed in early 2026 to provide an AI learning coach embedded in Moodle course pages, with multi-provider support, voice, analytics, and RAG over course content. Released to the wider Moodle community at https://github.com/saylordotorg/moodle-local_ai_course_assistant.

## Branding
The "SOLA" short name and "Saylor Online Learning Assistant" display name ship as the default for Saylor deployments. All institution-facing strings (institution name, assistant display name, short name, welcome message, chat greeting, avatar colours) are configurable in Site admin > Plugins > Local plugins > AI Course Assistant, so any institution can fully white-label the plugin without touching code.
